**Who:** me and 2 other second year students

**What:** Created an interactive 3D world with procedurally generated terrain, stylized rendering, and a custom UI

**When:** December 2021

**Where:** Brown University

**Why:** Intro Computer Graphics final project

**How:** `C++`, `OpenGL`, `Qt`

</OutlineBox>

<div>

Features:

-   Stylized rendering in a cell shaded style using a custom lighting model
-   Variable ink drawing style outlines generated using silhouette detection
-   Multiple types of terrain generation using value noise and Fractional Brownian Motion
-   Pseudo randomly generated trees dispered throughout terrain created using custom primitive meshes
-   Camera movement that allows user to traverse the scene immersively using keyboard controls

</div>
